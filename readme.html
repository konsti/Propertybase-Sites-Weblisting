XML Webservices on Sites
Version 1.5

Installation

  1. Unzip the package in an empty directory.
  2. Open up config/settings.php with a text editor like WordPad or similar and fill in your salesforce URL and your security token
  3. Save the file.
  4. Upload everything to your webserver.
  5. You can open search.php and enjoy your new weblisting, or create new accounts with lead_form.html.

Customization

    Search form (search.php):

  • Edit the __displaySearchForm function, or replace it with a template engine to change the search form.
  • Adjust the MAXIMUM_BEDROOMS constant in config/settings.php if you change the bedrooms search field.

    • Listing Result (listing.php):

  • Fields available in the salesforce response ($xml): Adjust the LISTING_FIELDS constant in config/settings.php.
  • Edit the sort order: Adjust LISTING_ORDER_BY constant in config/settings.php.
  • Edit the __displayTable function, or replace it with a template engine to adjust the listing layout and design.

    • Property Detail View (detail.php):

  • Fields available in the salesforce response ($xml): Adjust the DETAIL_FIELDS constant in config/settings.php.
  • Edit the __displayProperty function, or replace it with a template engine to adjust the listing layout and design.

System Recommendations

  • PHP version 5.2 or higher.
  • PHP SimpleXML extension.
  • PHP setting allow_url_fopen
